== Description ==

Simplicity is easy to use gallery plugin that allows you to create and manage image galleries just dropping the files into a folder of your choice. There is no complicated interfaces, a lot of unknown options and questions like "why my images are not shown". It's really simple, just upload the images into a folder with write permissions for the server. This is it!

== Installation ==

INSTALLATION: The easiest way to enjoy Simplicity Gallery is to login to you WordPress dashboard, go to Plugins >> Add New, search for Simplicity Gallery, and click to install. You can also download the zip file from this page and upload it from the Plugins >> Add New > Upload page.

HOW TO USE: Just go to a page or post and bellow the editor window you will see the Simplicity configurations. If you want to attach a gallery to your post or a page just type the name of the images folder where your files are located. Please note that directory should be accessible from the web server to write.

DOCUMENTATION: Just check <a href="http://saiman.thefreeart.com/simplicity-gallery" target="_blank">Simplicity Gallery Documentation</a> for more information regarding options.

== Frequently Asked Questions ==

= Do you have documentation or tutorials? =

Yes! See <a href="http://saiman.thefreeart.com/simplicity-gallery" target="_blank">Simplicity Gallery Documentation</a>.

= Will Simplicity Gallery work with my theme? =

Yes! Simplicity Gallery works with the vast majority of premium and free themes, without requiring any coding.

= Are the galleries flash based? =

No, Simplicity Gallery uses Javascript based displays to ensure compatibility across the widest range of displays possible.

= Are the galleries mobile friendly? =

Yes, since we use Javascript rather than flash, Simplicity Gallery is compatible with Android, iOS, and Blackberry.

= Can I modify the thumbnails size? =

Yes, we provide such an option so you can adjusted to suit your needs.

= Can I enable or disable the pagination? =

Yes, we are very flexible in this point of view, so you can configure the pagination depending your needs.
